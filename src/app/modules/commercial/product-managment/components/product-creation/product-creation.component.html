Creación de Producto

Ingrese la información requerida para crear un nuevo producto:

El campo Nombre es requerido.
El campo Código es requerido.
El campo Descripción es requerido.
El campo Cantidad Mínima es requerido.
La Cantidad Mínima no puede ser negativa.
El campo Cantidad Máxima es requerido.
El campo Porcentaje de IVA es requerido y debe estar entre 0 y 100.
El campo Unidad de Medida es requerido.
El campo Proveedor es requerido.
El campo Categoría es requerido.
El campo Precio es requerido y debe estar entre 0 y 100.
